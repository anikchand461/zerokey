Zerokey CLI

Installation

Using npm

npm install -g zerokey-cli

Using pip

pip install zerokey-cli

Using Go

go install github.com/anikchand461/zerokey-cli@latest

Verify Installation

zerokey --version

Available Commands

$ zerokey login
Authenticate with your ZeroKey account
$ zerokey add <name> <key>
Add a new API key with custom name
$ zerokey get <name>
Retrieve an API key by name
$ zerokey list
Display all stored API keys
$ zerokey delete <name>
Remove an API key from storage
$ zerokey sync
Synchronize keys with cloud
$ zerokey usage <name>
View usage statistics
$ zerokey export
Export all API keys
$ zerokey config
Configure CLI settings
