Available Commands

$ zerokey login
Authenticate with your ZeroKey account
$ zerokey register
Create your ZeroKey account
$ zerokey add-key
Add a new API key
$ zerokey ls
Display all stored API keys
$ zerokey delete <sl. no>
Remove an API key from storage
$ zerokey call <unified-api-key>
Test the endpoint
$ zerokey usage
View usage statistics
