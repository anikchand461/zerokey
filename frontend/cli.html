zerokey

Installation

Using npm

npm install -g zerokey-cli

Using pip

pip install zerokey-cli

Using Go

go install github.com/anikchand461/zerokey-cli@latest

Verify Installation

zerokey --version

CLI Features

Quick Search

Search and retrieve API keys instantly from your terminal

Secure Storage

Encrypted local storage for all your API credentials

Auto-Complete

Tab completion for commands and key names

Cross-Platform

Works seamlessly on Linux, macOS, and Windows

Available Commands

$ zerokey login
Authenticate with your ZeroKey account using username and password or OAuth providers
$ zerokey add <name> <api-key>
Add a new API key with a custom name for easy retrieval. Supports optional expiration dates.
$ zerokey get <name>
Retrieve an API key by name. Copies to clipboard automatically for quick use.
$ zerokey list
Display all stored API keys with their names, providers, and expiration dates in a formatted table.
$ zerokey delete <name>
Remove an API key from storage. Requires confirmation to prevent accidental deletion.
$ zerokey sync
Synchronize your API keys with the cloud. Ensures your keys are backed up and accessible across devices.
$ zerokey usage <name>
View usage statistics and analytics for a specific API key including request counts and timestamps.
$ zerokey export [--format json|csv]
Export all API keys to a file in JSON or CSV format for backup or migration purposes.
$ zerokey config
Configure CLI settings including default output format, auto-sync, and clipboard timeout.
